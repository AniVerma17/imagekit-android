focus

fun focus(focusType: FocusType): ImagekitUrlConstructor

Method used to specify the focus which is coupled with the extract type of crop mode (crop mode is not needed if you are using auto focus) to get the area of the input image that should be focussed on to get the output image.

Parameters

focusType - Accepts value of type FocusType. Possible values include center, top, left, bottom, right, top_left, top_right, bottom_left, bottom_right and auto. Default value - center
