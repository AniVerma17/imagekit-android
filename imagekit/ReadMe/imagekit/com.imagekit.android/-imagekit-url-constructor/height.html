fun height(height: Float): ImagekitUrlConstructor

Method to specify the height of the output image.

Parameters

height - Accepts integer value greater than 1 and if a value between 0 and 1 is specified, then it acts as a percentage height. For eg, 0.1 means 10% of the original height, 0.2 means 20% of the original height.

Return
the current ImagekitUrlConstructor object.
