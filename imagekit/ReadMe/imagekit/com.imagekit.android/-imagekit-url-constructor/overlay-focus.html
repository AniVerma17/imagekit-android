fun overlayFocus(overlayFocus: OverlayFocusType): ImagekitUrlConstructor

Method used to specify the position of the overlaid image relative to the input image.

Parameters

overlayFocus - Possible values include center, top, left, bottom, right, top_left, top_right, bottom_left and bottom_right. Default value - center

Exceptions

ImagekitException - if not a single overlay is specified or the overlay coordinates have already been applied using either overlayPosX() or overlayPosY()
