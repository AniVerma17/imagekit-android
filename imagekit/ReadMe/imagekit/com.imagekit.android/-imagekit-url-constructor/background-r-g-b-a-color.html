backgroundRGBAColor

fun backgroundRGBAColor(backgroundRGBAColor: String): ImagekitUrlConstructor

Method used to specify the background color as an RGBA code (e.g. FFAABB50) to be used for the image.

Parameters

backgroundRGBAColor - If you specify an 8-character background, the last two characters should be numbers from 00 to 99 which indicate the opacity level of the background. 00 corresponds to an opacity of 0.00, 01 corresponds to an opacity of 0.01 and so on. Default value - Black 000000

Exceptions

ImagekitException - if color is not a valid RGBA code.

Return
the current ImagekitUrlConstructor object.
