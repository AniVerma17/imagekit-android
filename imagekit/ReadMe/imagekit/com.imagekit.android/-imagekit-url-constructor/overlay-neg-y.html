overlayNegY

fun overlayNegY(overlayNegY: Int): ImagekitUrlConstructor

Method used to provide more granular control over the positioning of the overlay image on the input image. Negative values are supported with a leading capital N in front of the value provided. The value provided is subtracted from the original dimension of the image & positioned accordingly.

Parameters

overlayNegY - Possible values include integers less than zero.

Exceptions

ImagekitException - if not a single overlay is specified or the overlay focus has already been applied or overlayPosY is not a negative integer.

Return
the current ImagekitUrlConstructor object.
