overlayPosY

fun overlayPosY(overlayPosY: Int): ImagekitUrlConstructor

Method used to provide more granular control over the positioning of the overlay image on the input image. The top left corner of the input image is considered as (0,0) and the bottom right corner is considered as (w, h) where w is the width and h is the height of the input image.

Parameters

overlayPosY - Possible values include zero and positive integers.

Exceptions

ImagekitException - if not a single overlay is specified or the overlay focus has already been applied or overlayPosY is less than 0.

Return
the current ImagekitUrlConstructor object.
