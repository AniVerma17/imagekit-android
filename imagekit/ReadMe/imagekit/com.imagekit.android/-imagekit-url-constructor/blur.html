blur

fun blur(blur: Int): ImagekitUrlConstructor

Method to specify the Gaussian blur that has to be applied to the image. The value of blur decides the radius of the Gaussian blur that is applied. Higher the value, higher is the radius of Gaussian blur.

Parameters

blur - Accepts integer value between 1 and 100.

Exceptions

ImagekitException - if quality is outside the 1 to 100 range.

Return
the current ImagekitUrlConstructor object.
