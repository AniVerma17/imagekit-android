border

fun border(borderWidth: Int, borderColor: String): ImagekitUrlConstructor

Method used to specify the width and color of the border that is added around the image. The width is a positive integer that specifies the border width in pixels. The border color is specified as a standard RGB hex code e.g b-{width}_{color}

Parameters

borderWidth - width of the border

borderColor - color of the border as RGB hex code

Exceptions

ImagekitException - if color is not a valid RGB hex code.

Return
the current ImagekitUrlConstructor object.
